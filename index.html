About

Two disciplines, one eye for detail

I'm a software developer based in India who picks up new frameworks and languages quickly — whatever the project needs, I learn it and ship with it. My hands-on experience spans ASP.NET Web Forms, Laravel, PHP, Python, and Web APIs, backed by solid work with SQL Server and MySQL on the data side.

Outside the editor, I draw. Give me a photograph and I'll hand you back a graphite portrait that actually looks like the person — proportions, light, the small asymmetries that make a face recognizable. I post finished pieces on Instagram under @vishal_kalakaar.

Whether you need a working web application or a portrait that captures someone the way they really look, I bring the same approach: study the reference closely, then build it up carefully, layer by layer.
